# TesloDB API

1. Clone Proyect

2. ``` yarn  install ```

3. Clone the archive ```.env.template``` and rename to ___ .env ___

4. Fill the environment variables

5. Build database
```
docker-compose up -d
```

6. Build: ``` yarn start:dev ```
